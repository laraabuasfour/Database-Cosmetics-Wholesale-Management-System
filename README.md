# Database-Cosmetics-Wholesale-Management-System

This project was developed as a **Database Project** for the **Database Management Systems course (ENCS333)** at **Birzeit University**.

---

## Project Overview

This system is designed to manage a **cosmetics wholesale business**, including products, suppliers, warehouses, employees, and customer orders.

The system helps the wholesaler:
- Track product quantities and expiration dates  
- Manage suppliers and purchases  
- Handle customer orders and sales  
- Monitor warehouses and inventory  
- Generate bills for transactions  

The application integrates a **JavaFX graphical interface** with a **MySQL relational database**.

---

## Technologies Used

- Java  
- JavaFX  
- MySQL  
- SQL  
- Maven  
- IntelliJ IDEA  

---

## Database Design

The database was designed based on a real-world scenario and includes multiple entities:

- Products  
- Suppliers  
- Stores  
- Orders  
- Order Details  
- Warehouses  
- Employees  
- Vehicles  
- Bills  

Relationships include:
- Suppliers provide products  
- Stores place orders  
- Orders contain multiple products  
- Warehouses store products  
- Employees manage operations  

---

## Features

- Add / update / delete products  
- Manage suppliers and purchases  
- Handle store orders and sales  
- Track warehouse inventory  
- Employee and driver management  
- Billing system for transactions  
- User-friendly GUI using JavaFX
